# M-Tron Bass Setup

## Preset

Double Bass Wide Basic (Factory Sound Library > Basses)

## Track Setup

1. Create Instrument Track
2. Add M-Tron Pro IV
3. Load Double Bass Wide Basic
4. Add Note FX: Humanize (timing 5%, velocity 10%)
5. Add FX: EQ (low cut 40Hz, slight boost 800Hz)
6. Add FX: Compressor (4:1, medium attack)
7. Add FX: Tape Saturation (subtle warmth)
8. Add FX: Short Reverb (room, 0.5s, 15% wet)

## MIDI File

mtron-bass.mid

## Contents

- 8 bars matching picked acoustic chord progression
- Bars 1-2: Am-G root-fifth pattern
- Bars 3-4: F-E root-fifth pattern
- Bars 5-6: Am-C-G-D busier movement
- Bars 7-8: Am whole notes
- CC1 builds 40→50→60→70 through sections
- Note range: B1-A2
- No tempo (set via transport)
